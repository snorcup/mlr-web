How to use

1. Load audio

Click Start audio, then drag audio files onto track slots or click a track to pick files. Each track plays one clip at a time.

2. REC view (default)

Tap a track row to set focus. Use columns: Rec (arm), Focus (select), Tempo (map), Rev (reverse), Speed (x9–x16), Stop/Start (x16). Alt+focus toggles tempo map.

3. CUT view

Tap any pad in track rows to jump to that slice. Two-finger press sets a loop region (first finger = start, second = end). Alt+press toggles start/stop.

4. CLIP view

Select a clip (x=1–7) for each track. Each track can point to any of 16 clips. Clips are independent audio buffers.

5. Patterns

Press a REC button (P1–P4), perform slice hits, press again to stop. Press PLAY to loop the captured gesture. Alt+pattern button clears.

6. Nav row

Top row: x=1 REC, x=2 CUT, x=3 CLIP, x=5–8 patterns, x=9–12 recalls, x=15 quantize (hold for TIME), x=16 alt. Alt+x=1 clears buffer.

7. Connect monome

Click Connect monome USB. The browser talks to the local serialosc bridge. Requires a Chromium browser over HTTPS or localhost.
