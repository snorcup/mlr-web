How to use

1. Load audio

Click Start audio, then drag audio files onto track slots or click a track to pick files. Each track plays one clip at a time.

2. Trigger loops

Tap any pad in rows 1–7. The clip loops from that slice. Tap the same pad again to stop that track. Tap a different pad on the same track to jump to a new slice.

3. View modes

CUT — default trigger mode. REC — same triggers, use with pattern record buttons. TIME — press a track pad for loop start, press again for loop end. Playback loops within that region.

4. Patterns

Press a REC button (P1–P4), perform slice hits, press again to stop. Press PLAY to loop the captured gesture. Patterns record with precise timing and loop independently.

5. Per-track modes

Bottom row columns 5–8 set per-track behavior. Press a mode button (CUT, SOLO, MUTE, or ONCE), then tap a track row (1–7) to apply it. The mode button lights up while pending. Press it again to cancel.

CUT — default toggle loop/stop. SOLO — mutes all other tracks. MUTE — silences the track. ONCE — plays through once without looping. Track badges (S, M, 1) show the active mode per track.

6. Stop all

Press STOP ALL (bottom row, column 4) to cut everything at once. The button lights up when any track is playing.

7. Connect monome

Click Connect monome USB. The browser talks to the local serialosc bridge. Requires a Chromium browser over HTTPS or localhost.
