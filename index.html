How to use MLR Web

A browser-based port of tehn/mlr — live sample cutting for monome grids.

Getting started

  • Press Start audio to initialize the Web Audio context (required before anything plays).
  • Press Connect monome USB to connect your grid.
  • Load samples by clicking a track slot or dragging audio files onto it.
  • The app starts in CUT view — press pads on track rows (y=1–6) to trigger playback.

Grid layout

  • Row 0 (top) — Nav: view buttons (x=0-2), patterns (x=4-7), recalls (x=8-11), quantize (x=14), alt (x=15)
  • Rows 1–6 — Tracks 1–6, columns = 16 slice positions
  • Row 7 (bottom) — Modes (x=0-3: CUT/SOLO/MUTE/ONCE), patterns (x=4-7)

CUT view (default)

Main performance view:

  • Tap a pad — Start looping from that slice position
  • Tap same pad again — Stop that track (toggle)
  • Tap different pad — Jump to new position, keep playing
  • Two-finger press — Set loop region (release to confirm)
  • Bright dot = playhead, dim region = loop area

REC view

Per-track parameter control:

  • x=0 — Record arm
  • x=2,3 — Focus select (Alt = tempo map toggle)
  • x=7 — Reverse
  • x=8–14 — Speed (x=11 = 1×)
  • x=15 — Start / Stop track

Track modes (bottom row x=0–3)

Press a mode button, then press a track row to apply:

  • CUT (x=0) — Normal: tap to loop, tap again to stop
  • SOLO (x=1) — Mutes all other tracks
  • MUTE (x=2) — Silences the track
  • ONCE (x=3) — One-shot: plays through once without looping

Press the same mode button again to cancel pending assignment.

Patterns (nav x=4–7, bottom x=4–7)

  • Empty — Press to start recording
  • Recording — Press again to stop recording and start playback
  • Playing — Press to stop
  • Has data — Press to play
  • Alt + press (nav) — Clear pattern

One button per pattern. Nav row and bottom row mirrors the same 4 patterns.

Recalls (nav x=8–11)

  • 1st press — Start recording all events
  • 2nd press — Stop recording
  • 3rd press — Replay captured events
  • Alt + press — Clear

Quantize & Alt

  • Nav x=14 — Toggle quantize (hold events until clock tick)
  • Alt + x=14 — Enter TIME view
  • Nav x=15 — Alt modifier (hold for alternate functions)

Monome connection

Two connection methods, tried in order:

  1. Bridge (WebSocket) — localhost:8089. Run serialosc-bridge on the machine with the monome.
  2. Web Serial — Direct USB via Chrome/Edge (HTTPS or localhost required).

If the monome shows connected but pads do nothing, make sure audio has been started.
